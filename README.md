# 🔥 DailyFire

A personal task scheduler and daily routine tracker built to keep you accountable, motivated, and consistent every single day.

## Live App
- **Frontend:** https://dailyfire.netlify.app
- **Backend:** https://dailyfire.onrender.com

## Features
- ✅ Daily routine tracker with 16 pre-loaded weekday tasks
- ✅ Add, edit and delete routine tasks
- ✅ One-off task list with priority levels and due dates
- ✅ XP system and level progression
- ✅ Streak tracking with badges and rewards
- ✅ Daily progress ring and motivational messages
- ✅ Celebration animation on full day completion
- ✅ Weekly badge rewards

## Tech Stack
- **Frontend:** React + Vite + TypeScript + Framer Motion
- **Backend:** Python FastAPI
- **Database:** MongoDB Atlas
- **Hosting:** Netlify (frontend) + Render (backend)

## Deployment
Frontend auto-deploys from GitHub via Netlify.
Backend runs on Render free tier — may take 30 seconds to wake up if inactive.
